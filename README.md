# Coffee Delivery

Desafio Rocketseat Ignite  
Aplicação para gerenciar um carrinho de compras de uma cafeteria fictícia  

## Stack

React 19 · TypeScript 5 · Vite 7 · styled-components 6 · React Router 7 · React Hook Form + Zod 4 · Immer · ESLint 9 (flat config)

## Como rodar

```
git clone https://github.com/m4cm3nz/coffee-delivery-react-ts.git
cd coffee-delivery-react-ts
npm i
npm run dev
```

Outros scripts: `npm run build` (type-check + build de produção), `npm run preview`, `npm run lint`.
